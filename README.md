# Simulação de Revisão Espaçada

Este repositório contém o código utilizado no Trabalho de Conclusão de Curso (TCC) para a realização de simulações computacionais de revisão espaçada, com foco na análise do desempenho do modelo HLR, considerando a habilidade dos estudantes e a complexidade das questões.

# Objetivos

Comparar, por meio de simulações controladas, o desempenho do modelo HLR, analisando:
- A taxa de acerto durante a prática;
- A retenção de conhecimento ao longo do tempo;
- A frequência de fallback;
- O impacto dos algoritmos e dos horizontes temporais na prática de revisão espaçada.

#  Metodologia

As análises foram conduzidas com alunos simuladas, permitindo o controle das variáveis:
- Habilidade dos estudantes;
- Complexidade das questões;
- Horizontes temporais
- Limiares de retenção

Os resultados refletem o comportamento definido pelo modelo computacional adotado e não representam diretamente contextos educacionais reais.
